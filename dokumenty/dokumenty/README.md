### 📂 **Формат файлов для GitBook**

* Тексты — Markdown (`.md`)
* Таблицы/сметы — `.xlsx` или `.csv` + ссылка/вставка
* Чертежи — `.svg`, `.png`, `.pdf`
* Документы — `.pdf`, `.docx` (если вложения нужны)
* Видео/интерактив — ссылка на YouTube, Loom, Vimeo
