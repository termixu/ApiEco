# Устройство

## 🐝 Устройство Flow Hive: как работает технология слива мёда

### 🧱 1. Общая конструкция улья

Flow Hive — это модифицированный **Langstroth-ульеобразный улей**, где верхняя надставка (Flow Super) оснащена **умной рамкой Flow Frame**, позволяющей собирать мёд **без открытия улья** и **без разрушения сот**.

#### Основные части:

| Компонент                                    | Описание                                                     |
| -------------------------------------------- | ------------------------------------------------------------ |
| 📦 **Brood Box (корпус)**                    | Нижняя часть улья, где находится расплод и матка             |
| 🥄 **Flow Super (верхняя надставка)**        | Где находится Flow Frame и собирается мёд                    |
| 🧱 **Flow Frame**                            | Рамка с поточными ячейками, через которые можно собирать мёд |
| 🗝️ **Flow Key (ключ управления)**           | Поворачивает механизм внутри рамки                           |
| 🪵 **Нижняя доска / Floor Board**            | Защита от мышей, муравьёв, контроль влажности                |
| 🌬️ **Вентиляционная крышка или Slope Roof** | Поддержание микроклимата внутри улья                         |
| 🪟 **Прозрачное окно**                       | Для наблюдения за уровнем мёда                               |
| 🚰 **Трубка слива мёда**                     | Через неё мёд вытекает в банку                               |

📌 Это **не просто улей. Это система**, которая позволяет собирать мёд **естественным образом**, без стресса для пчёл.

***

### 🧩 2. Что такое Flow Frame?

**Flow Frame** — сердце системы. Это **пластиковая рамка**, заполненная сотами, которые можно **раздвигать изнутри**, чтобы мёд стёк сам.

#### Внутреннее устройство:

* **Ячейки разделены пополам**
* **Половинки ячеек соединены по центральной оси**
* При повороте ключа половины **раздвигаются**, образуя **вертикальные каналы**
* Эти каналы направляют мёд вниз
* Внизу есть **желобок**, который направляет мёд в трубку

📌 Это как **тайный водопровод**, только вместо воды — **мёд**, и **никто не должен покидать дом ради него**.

***

### 🔑 3. Механизм поворота ячеек

#### Как работает Flow Key:

1. Ключ вставляется в боковую часть рамки
2. При повороте он активирует **центральный вал**
3. Вал связан с **тонкими лопастями внутри сот**
4. Лопасти двигают половинками ячеек
5. Ячейки **раздвигаются**, создавая щель
6. Мёд начинает **стекать по этим щелям**
7. После сбора ты возвращаешь ключ обратно → соты снова становятся целыми

📌 Этот процесс занимает **минимум усилий**, но **дает максимум результата**.

***

### 🍯 4. Как происходит слив мёда?

#### Пошагово:

1. 🐝 Пчёлы заполняют ячейки мёдом и **запечатывают их воском**
2. 🧠 Ты проверяешь уровень мёда через **прозрачное окно**
3. 🥄 Повернув ключ, ты **открываешь внутренние каналы**
4. 🕊️ Мёд начинает **медленно стекать по щелям**
5. 🧾 Он собирается в **желобке внизу рамки**
6. 🧬 Через **трубку мёд выходит в банку**
7. 🥄 После сбора ты возвращаешь ключ в исходное положение\*\*
8. 🌿 Пчёлы продолжают работать, как ни в чём не бывало\*\*

📌 Это как **водопровод в природе**: ты открываешь кран — и **мёд течёт сам**, без насильственного отбора.

***

### 🧭 5. Почему пчёлы почти не реагируют?

* 🐝 Пчёлы работают **в нижней части улья**
* 🧠 Flow Frame находится **наверху**, там, где хранится мёд
* 🕊️ Они не видят, что произошло изменение ячеек
* 🧾 Не теряют связи с семьей
* 🧬 Просто наблюдают, что **мёд стал меньше**
* 🥄 Продолжают достраивать соты, как обычно

📌 Это **не нарушает порядок**, а **работает с ним**, используя силу тяжести и естественные законы.

### 🧬 7. Что происходит внутри Flow Frame?

#### Устройство ячейки:

* Каждая ячейка состоит из **двух половинок**
* Они **не склеены**, а **соединены механически**
* При повороте ключа они **раздвигаются**, образуя **каналы**
* Эти каналы превращают ячейки в **естественный мёдопровод**

📌 Это как **система «открыл — и пусть течёт»**, только для мёда.

### ⚠️ 10. Возможные слабые места Flow Hive

| Проблема                                      | Почему возникает                         | Как улучшить                                                      |
| --------------------------------------------- | ---------------------------------------- | ----------------------------------------------------------------- |
| 🧱 **Пластиковые соты**                       | Могут перегреваться, терять эластичность | Использовать **биоматериалы** или **дерево + воск**               |
| 🥄 **Слив мёда требует внимания**             | Если оставить слишком долго — забродит   | Сделать **автоматический слив** или **индикатор уровня зрелости** |
| 🧾 **Пчёлы могут начать строить соты внутри** | Из-за долгого хранения мёда              | Добавить **ароматические барьеры**                                |
| 🧬 **Засор трубки**                           | Из-за воска или пыли                     | Сделать **съёмную трубку** или **фильтр из ткани**                |
| 🌿 **Не все породы принимают Flow Frame**     | Некоторые пчёлы игнорируют его           | Добавить **ароматическую навигацию**                              |
| 🧠 **Сложно чистить**                         | Нужно часто протирать трубки             | Сделать **самоочищающуюся систему**                               |
| 🧃 **Высокая стоимость**                      | Пластик и технологии делают его дорогим  | Создать **эко-альтернативу** из дерева и тростника                |
| 🌞 **Перегрев в жару**                        | Пластик может деформироваться            | Добавить **деревянные вставки** или **терморегуляторы**           |
| 🧾 **Ограниченная совместимость**             | Не подходит всем ульям                   | Сделать **универсальные рамки**                                   |
| 🧬 **Нет фильтрации мёда**                    | Может содержать мусор                    | Добавить **съёмный фильтр из ткани** или **древесного угля**      |

📌 Ты можешь **взять лучшее из Flow Hive**, и **создать своё решение**, которое будет **ещё более экологичным, простым и доступным**.
