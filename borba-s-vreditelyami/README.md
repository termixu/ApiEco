# 🛡️ Борьба с вредителями

Пасека — это не только мир пчел, но и поле битвы с вредителями: паразитами, грызунами и болезнями. Знание врагов и методов борьбы — ключ к сохранению здоровья семей.

***

## **1. Основные вредители пасеки**

**а) Паразиты**

| **Вредитель**                                                             | **Угроза**                                                 | **Методы борьбы**                                             |
| ------------------------------------------------------------------------- | ---------------------------------------------------------- | ------------------------------------------------------------- |
| **Варроа destructor**                                                     | Паразитирует на пчелах, переносит вирусы, ослабляет семью. | **Химические:** Апибио, Апилан, Формикул.                     |
| **Механические:** Сетка против варроа, тепловой метод (40°C на 15 минут). |                                                            |                                                               |
| **Натуральные:** Эфирные масла (чабрец, тимьян), прополис.                |                                                            |                                                               |
| **Моль**                                                                  | Ест воск и пыльцу, разрушает соты.                         | **Химия:** Пинцивецин.                                        |
| **Натуральные:** Сушка рамок на солнце, хранение в герметичных ящиках.    |                                                            |                                                               |
| **Муравьи**                                                               | Крадут мед, нападают на пчел.                              | **Барьеры:** Солевая соль, масляные пропитки для ножек ульев. |
| **Ловушки:** Тарелки с водой и мылом.                                     |                                                            |                                                               |

**б) Животные**

| **Вредитель**                                           | **Угроза**                       | **Методы борьбы**                                           |
| ------------------------------------------------------- | -------------------------------- | ----------------------------------------------------------- |
| **Мыши**                                                | Разрушают ульи, едят пчел и мед. | **Сетки на входе** (отверстия 3–4 мм).                      |
| **Мышеловки** (без ядов, чтобы не убить пчел).          |                                  |                                                             |
| **Утепление ульев** — мыши любят тепло.                 |                                  |                                                             |
| **Шершни**                                              | Убивают пчел, грабят мед.        | **Ловушки с сахарным сиропом** (шершни погибают в ловушке). |
| **Уничтожьте гнезда** шершней в радиусе 1 км от пасеки. |                                  |                                                             |

**в) Биологические угрозы**

| **Вредитель**                                  | **Угроза**                                     | **Методы борьбы**                                        |
| ---------------------------------------------- | ---------------------------------------------- | -------------------------------------------------------- |
| **Нозематоз**                                  | Грибковое заболевание, приводит к гибели пчел. | **Антибиотики:** Фумидил, Фитоверм.                      |
| **Кормление прополисом** — повышает иммунитет. |                                                |                                                          |
| **Голодная болезнь**                           | Падеж пчел из-за нехватки корма.               | **Кормление сахарным сиропом** (1:1) в периоды цветения. |

***

## **2. Сезонная борьба с вредителями**

**а) Весна**

* **Варроа:** Первичная обработка препаратами (Апибио).
* **Муравьи:** Установите соляные кольца вокруг ульев.

**б) Лето**

* **Варроа:** Ежемесячный контроль с помощью белой бумаги под ульем.
* **Шершни:** Устанавливайте ловушки с сиропом или фруктами.

**в) Осень**

* **Моль:** Сушите рамки под солнцем, храните в герметичных ящиках.
* **Мыши:** Установите сетки на входе и мышеловки.

**г) Зима**

* **Влажность:** Убедитесь в вентиляции — грибки разрушают соты.
* **Мышей:** Проверяйте ульи раз в месяц, добавляйте утеплитель.

***

## **3. Топ-5 методов борьбы**

**1. Профилактика — лучшая защита**

* **Чистота пасеки:** Убирайте старые рамки и восковую грязь.
* **Регулярные проверки:** Каждые 2 недели осматривайте ульи.

**2. Механические методы**

* **Сетка против варроа:** Устанавливайте пластину с отверстиями 4,2 мм — паразиты выпадают.
* **Тепловой метод:** Нагревайте улей до 40°C для уничтожения варроа.

**3. Химия с умом**

* **Апибио:** Эффективен против всех стадий варроа.
* **Пинцивецин:** От моли, но используйте только в отсутствие пчел.

**4. Природные средства**

* **Эфирные масла:** Чабрец или тимьян добавляют на рамки — отпугивают варроа.
* **Прополис:** Пчелы сами его используют как антисептик.

**5. Защита от грызунов**

* **Сетки на входе:** Закройте улей сеткой с отверстиями 3 мм — мыши не пройдут.
* **Мышеловки:** Используйте без ядов (например, «пленка-ловушка»).

***

## **4. Советы экспертов**

* **«Не доверяй тишине»:**
  * Если пчелы перестали летать — откройте улей: возможно, варроа или голод.
* **«Следи за соседями»:**
  * Если на пасеке рядом — договоритесь о совместной борьбе с варроа.
* **«Солнце — лучший дезинфектор»:**
  * Сушите рамки под солнцем — убивает грибки и яйца паразитов.

***

## **5. Примеры из практики**

* **Победа над варроа:**
  * Пчеловод Иван сочетал Апибио с тепловым методом — уничтожил паразитов на 90%.
* **Борьба с мышами:**
  * Ольга установила сетки и мышеловки — грызунов не было годами.

***

## **6. Частые вопросы (FAQ)**

**«Как узнать, есть ли варроа?»**

* Положите белый лист под ульем: если видите красные точки (клещи) — варроа есть.

**«Можно ли использовать яды от мышей?»**

* Нет! Яды опасны для пчел. Используйте ловушки или сетки.

**«Почему муравьи атакуют улей?»**

* Привлекает запах меда. Уберите остатки меда у входа в улей.

***

## **7. Противогрибковые меры**

* **Влажность:** Удерживайте её ниже 60% (используйте вентиляцию).
* **Прополис:** Добавьте рамки с прополисом — он защищает улей.
* **Стерилизация:** Кипятите снаряжение перед сезоном.

#### **Заключение: Защита — это наука и забота**

Борьба с вредителями требует внимания и дисциплины. Но если применять методы правильно — пасека будет процветать. Как говорил один старый пчеловод: «Хороший пчеловод — тот, кто видит врага до того, как он придет». 🐝⚔️

P.S. Если заметили, что пчелы летят на соседний участок — проверьте, нет ли варроа или грибка в вашем улье!
