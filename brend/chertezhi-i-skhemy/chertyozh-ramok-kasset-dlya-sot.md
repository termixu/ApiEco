# Чертёж рамок/кассет для сот

### 📏 **Основные параметры**

* **Форма**: прямоугольная или овальная (подходит под корпус улья)
* **Материал**: лиственница, сосна, покрытые био-лаком
* **Толщина планок**: 2–3 см
* **Высота рамки**: 30–40 см
* **Ширина рамки**: 25–35 см
* **Внутреннее пространство**: для вощины или восковых сот с капиллярным каналом

***

### 🧩 **Устройство рамки**

#### 1. **Каркас**

* **Горизонтальные планки** сверху и снизу (опора для сот)
* **Вертикальные стойки** по бокам (толщина 2 см, с пазами для вощины)

#### 2. **Вощина / восковые соты**

* **Закладка**: вставляется в пазы рамки (без проволоки или пластика)
* **Канал слива**: вертикальный тростниковый канал по центру или краю рамки (от сота к желобку)

#### 3. **Механизм слива**

* **Тростниковый канал**: внутри рамки → мёд стекает по капиллярной системе
* **Ручка-ключ**: поворотная деревянная планка на верхней планке рамки → открывает канал слива

***

### 🍯 **Как работает?**

```
[Пчёлы заполняют соты мёдом]  
        ▼  
[Мёд достигает тростникового канала]  
        ▼  
[Поверни ключ → мёд начинает течь по тростнику в желобок]  
        ▼  
[Мёд собирается в банку, пчёлы продолжают работать]  
```

📌 **Без медогонки, без открытия улья. Мёд течёт сам**, как река.

***

### 🧾 **Ключевые особенности**

| Элемент                 | Описание                                                    |
| ----------------------- | ----------------------------------------------------------- |
| **Канал слива**         | Тростниковая трубка внутри рамки → мёд стекает без стресса  |
| **Крепление сот**       | Пазы в рамке → легко заменить, не повреждая структуру       |
| **Механизм управления** | Ручка-ключ на верхней планке → открывает/закрывает канал    |
| **Вентиляция**          | Щели между рамками → воздух циркулирует, не перегревая улей |
| **Съёмные соты**        | Можно извлечь для очистки или ремонта                       |

***

### 🪵 **Совместимость с ульем Mellifera**

* **Пазы в корпусе**: точно повторяют размер рамок
* **Смотровое окно**: позволяет видеть уровень мёда без открытия улья
* **Желобок под сливом**: направляет мёд в банку через боковое отверстие

***

### 🧠 **Принципы проектирования**

> _«Не разрушай соты. Не тревожь пчёл. Пусть мёд течёт сам»_
