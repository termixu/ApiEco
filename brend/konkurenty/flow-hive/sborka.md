# Сборка

## 🐝 Сборка улья Flow Hive

_Создай дом, из которого мёд течёт сам_

### 🧰 1. Подготовка: что понадобится

#### Инструменты:

| Инструмент                 | Зачем                               |
| -------------------------- | ----------------------------------- |
| 🔧 Шуруповёрт или отвёртка | Для закручивания болтов и креплений |
| 📏 Рулетка                 | Проверить размеры и зазоры          |
| 🪚 Ножницы по дереву / нож | Для обрезки материалов              |
| 🧽 Щётка                   | Чистка трубок и желобов             |
| 🧴 Клей (если нужен)       | Для герметизации щелей              |

#### Материалы:

| Материал                            | Что входит                            |
| ----------------------------------- | ------------------------------------- |
| 📦 **Корпус улья (Brood Box)**      | Где находится расплод                 |
| 🥄 **Flow Super**                   | Верхняя надставка с Flow Frame        |
| 🧱 **Flow Frame**                   | Сотовая рамка с механизмом слива мёда |
| 🗝️ **Flow Key**                    | Ключ для поворота ячеек               |
| 🌬️ **Вентиляционная крышка**       | Для регуляции воздуха                 |
| 🪵 **Нижняя доска / Floor Board**   | Защита от мышей и мусора              |
| 🪟 **Прозрачное окно (по желанию)** | Визуальный контроль уровня мёда       |

📌 Все компоненты можно приобрести отдельно или в наборе.\
Также возможна **самостоятельная модификация под другие типы ульев** (Langstroth, Dadant).

***

### 🛠️ 2. Пошаговая сборка улья

#### 📌 Шаг 1: Распаковка и проверка

* Распакуй все коробки
* Проверь наличие всех деталей:
  * Корпус
  * Flow Super
  * Flow Frame
  * Крышка
  * Ключ
  * Трубка слива мёда
* Проверь, нет ли повреждений

📌 Лучше всего собирать в сухую погоду, где нет риска попадания дождя на древесину.

***

#### 📌 Шаг 2: Сборка корпуса улья

1. **Поставь нижнюю доску**
   * Она служит основой улья
   * Убедись, что она прочная и чистая
2. **Установи Brood Box (корпус улья)**
   * Поставь его на доску
   * Проверь, чтобы не было перекосов
3. **Закрепи рамки внутри**
   * Вставь стандартные рамки с восковыми сотами
   * Не трогай Flow Frame пока — он будет сверху

📌 Используй лёгкое покрытие из льняного масла или экологической пропитки для защиты дерева.

***

#### 📌 Шаг 3: Установка Flow Super

1. **Подними Flow Super аккуратно**
   * Можно использовать **Super Lifter**, если есть
2. **Поставь её сверху на Brood Box**
   * Не дави — просто опусти
3. **Проверь, чтобы рамки были установлены вертикально**
   * Иначе могут быть проблемы с потоком мёда

📌 Flow Super — это **мёдовый этаж**. Он должен быть **лёгким, но прочным**.

***

#### 📌 Шаг 4: Установка Flow Frame

1. **Возьми Flow Frame за бока**
   * Не сгибай — конструкция хрупкая
2. **Аккуратно вставь в Flow Super**
   * Детали должны плотно прилегать
3. **Проверь, чтобы ключ легко входил**
   * Примерь ключ, не проворачивай ещё
4. **Убедись, что трубка слива свободна**
   * Не должно быть препятствий
5. **Проверь прозрачное окно**
   * Чтобы видеть уровень мёда

📌 Пчёлы начнут заполнять Flow Frame через 1–2 недели после установки.

***

#### 📌 Шаг 5: Как работает Flow Frame?

**Конструкция:**

* Ячейки разделены на две половинки
* При повороте ключа они раздвигаются
* Образуются каналы для мёда
* Мёд стекает вниз по желобкам
* Вытекает через трубку в банку

📌 Это как **тайный канал в замке**: ты открываешь его — и **мёд вытекает без войны**, без медогонки, без стресса.

***

#### 📌 Шаг 6: Установка Flow Key и трубки слива мёда

1. **Вставь ключ в паз рамки**
   * Он должен легко поворачиваться
2. **Проверь, нет ли заеданий**
   * Если ключ не вращается — проверь, не перекошена ли рамка
3. **Установи трубку слива**
   * Подсоедини её к выходному отверстию
   * Убедись, что она направлена в банку
4. **Проверь систему слива**
   * Можно пролить немного воды и увидеть, как она стекает

📌 После этого можно считать улей **собранным и готовым к жизни**.

***

### 🌳 3. Размещение улья

#### 📍 Где лучше ставить?

* **Не на прямом солнце** — пчёлы не любят перегрева
* **На ровной поверхности** — лучше использовать **ножки или подставку**
* **Не в сыром месте** — плесень опасна
* **Не на сквозняке** — пчёлы теряют силы
* **Рядом с источником воды** — пчёлы будут довольны

📌 Идеальная ориентация: **юго-восточная сторона**, полутень, защита от ветра.

***

### 🧾 4. Первый запуск и проба работы

#### 🐝 Что делать:

* Поставь улей на место
* Пчёлы сами найдут Flow Frame
* Через 2–4 недели можно проверять через окно
* Когда мёд заполнит рамки — пора собирать

📌 **Не торопи их**. Они знают, как работать.

***

### 🍯 5. Как впервые собрать мёд

#### 📌 Шаги:

1. **Подставь банку под трубку**
2. **Медленно поверни ключ на 90°**
3. **Подожди 5–15 минут** — мёд начнёт стекать
4. **Поверни ключ обратно**, когда мёд закончится
5. **Почисти трубку после каждого сбора**

📌 Это первый момент, когда ты видишь, **работает ли система**, и **понравился ли улей пчёлам**.

### 🧪 6. Возможные ошибки при сборке

| Ошибка                                | Почему возникает                   | Как исправить                                   |
| ------------------------------------- | ---------------------------------- | ----------------------------------------------- |
| 🔧 **Перекос Flow Frame**             | При неправильной установке         | Переставь рамку, убедись в правильном положении |
| 🕯️ **Заедание ключа**                | Из-за влажности или перекоса       | Почисти, смажь натуральным маслом               |
| 🧽 **Засор трубки**                   | Из-за воска или пыли               | Промывай водой или продувай                     |
| 🥄 **Нельзя наблюдать за мёдом**      | Отсутствие окна или грязное стекло | Установи прозрачную крышку или протри окно      |
| 🧾 **Пчёлы игнорируют Flow Frame**    | Может быть слишком много рамок     | Убери лишние, оставь только Flow Frame          |
| 🧬 **Мёд забродил**                   | Слишком долго не собирали          | Собирай своевременно                            |
| 🌞 **Пластиковые соты перегреваются** | На солнце                          | Установи тень или используй деревянные элементы |

📌 Эти ошибки помогут тебе **понять слабые места** и **улучшить улей в будущем**.

### 🧭 7. Как сделать сборку улья более простой?

| Идея                                | Реализация                                 |
| ----------------------------------- | ------------------------------------------ |
| 🧰 **Инструменты в комплекте**      | Включить шестигранный ключ и съёмник рамок |
| 📹 **Видеоинструкция**              | QR-код на улье — смотри и делай            |
| 🧶 **Механизм поворота без усилий** | Рычаг вместо ключа                         |
| 🧾 **Простая установка Flow Frame** | Без помощи специалиста                     |
| 🧬 **Автоматическая система слива** | Без ручного управления                     |
| 🥄 **Фильтр на выходе**             | Хлопчатобумажная сеточка                   |
| 🌿 **Эко-материалы**                | Дерево, пробка, тростник вместо пластика   |

📌 Эти идеи помогут тебе **не просто повторить Flow Hive, а улучшить его**, сделав **более доступным, простым и экологичным**.

### 🧠 8. Как применить это к созданию своего улья?

> _«Как Flow Hive упрощает сбор мёда, так и твой улей может упростить всю жизнь пчеловода»_

📌 Вот, что можно улучшить:

* **Деревянные Flow Frame** — экологичнее и теплее
* **Слив через капиллярную систему** — мёд течёт сам
* **Цветовая сигнализация уровня мёда** — не нужно открывать улей
* **Автоматический слив при заполнении** — меньше вмешательства
* **Система самоочищения** — пчёлы удаляют мусор сами
* **Ароматические вставки** — защита от клещей и грабителей
* **Механический рычаг вместо ключа** — удобнее управлять

📌 Это уже **не просто улей. Это улей будущего**, где **технология и природа идут вместе**.
