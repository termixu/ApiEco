# ⚙️ Инструменты

### 👕 **Защитный костюм**

**Состав**:

* **Наголовник с сеткой** → защищает лицо и шею
* **Куртка и штаны** из плотного хлопка → не привлекает пчёл (не открывайте улей часто)
* **Перчатки** (кожаные или нейлоновые) → не тревожь пчёл резкими движениями

📌 **Совет**:

> Выбирай **светлые тона**. Пчёлы реже атакуют светлые цвета. После работы **промывай костюм**, чтобы убрать запах мёда.

***

### 🧯 **Дымарь**

**Как работает**:

* Блокирует **феромоны тревоги** → пчёлы не атакуют
* Рассеивает **запах человека** → пчёлы не воспринимают тебя как угрозу

#### 🔥 **DIY-дымарь из банки**

* **Металлическая банка** (от тушенки или кофе)
* **Трубка** (диаметр 2–3 см, длина 30 см)
* **Сетка** на дне → искры не повредят улей
* **Пакля или сухие травы** → дым без химии

📌 **Совет**:

> Используй **лаванду или мяту** в дыме → пчёлы расслабляются, а не агрессируют.

***

### 🥄 **Медогонка**

**Как работает**:

* Центробежное вращение рамок → мёд вытекает, не разрушая соты
* **Ручная медогонка** → для 1–5 ульев
* **Электрическая медогонка** → для 20+ ульев

#### 🧼 **Уход за медогонкой**:

* После использования **промывай кипятком**
* Храни в **сухом месте** → не теряет силы, не ржавеет

📌 **Совет**:

> Если используешь капиллярный слив через тростник → **медогонка не нужна**. Мёд течёт сам.

### 🧾 **Советы по выбору и уходу**

#### 👕 **Защитный костюм**

* **Цвет**: светлый (пчёлы не атакуют)
* **Уход**: после работы **промывай водой**, убери запах мёда
* **Эко-альтернатива**: органический хлопок, лён, без химии

#### 🧯 **Дымарь**

* **Материалы для дыма**: пакля, сухие травы, старый воск
* **Осторожность**: не используй **резину или пластик**
* **Современные модели**: электрические дымари → точный контроль дыма

#### 🥄 **Медогонка**

* **Ручная** → для малой пасеки, без электроэнергии
* **Электрическая** → для масштаба, но требует питания
* **Чистка**: **кипяток, хлопковая ткань** → сохраняй чистоту без химии

📌 **Главный принцип**:

> _«Не тревожь пчёл. Они должны чувствовать, что ты — не враг, а партнёр»_

### ❓ **Ответы на частые вопросы**

#### Работа без костюма?

> Возможна, но **только с опытом**. Пчёлы реагируют на **движения и запах**. Если ты спокоен и не тревожишь их — они не атакуют.

#### Выбор дымаря?

> Для 1–5 ульев → **ручной дымарь из банки**\
> Для 20+ ульев → **электрический дымарь с регулировкой дыма**

#### Как сохранить медогонку?

> **Промывай кипятком**, храни в сухом месте\
> **Не используй химию** → пчёлы теряют связь с семьёй
