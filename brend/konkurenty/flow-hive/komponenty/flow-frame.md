# Flow Frame

## 🧱 Flow Frame: рамка, из которой мёд течёт сам

_Не разрушая дом, забирай его плоды_

### 📖 Введение

**Flow Frame** — это **умная рамка**, которая делает возможным **сбор мёда без медогонки**, без дымаря и без нарушения улья. Она сочетает **инженерию, природу и терпение**.

📌 Она работает так:

> _«Пчёлы заполняют соты. Ты поворачиваешь ключ. Мёд начинает течь. Пчёлы продолжают работать»_

***

### 🔧 1. Устройство Flow Frame

#### Основные части:

| Часть                     | Описание                                  |
| ------------------------- | ----------------------------------------- |
| 🧱 **Пластиковая основа** | Изготовлена из полипропилена (PP)         |
| 🕳️ **Сотовые ячейки**    | Предварительно разделены на две половинки |
| 🗝️ **Механизм поворота** | Ключ активирует систему лопастей внутри   |
| 🌬️ **Желобок внизу**     | Собирает мёд после слива                  |
| 🚰 **Трубка для слива**   | По ней мёд выходит в банку                |

📌 Это как **тайный канал**, который можно открыть одним движением руки.

***

### 🧩 2. Как устроены соты внутри?

#### Принцип работы:

* Каждая ячейка состоит из **двух половинок**
* Они соединены по центральной оси
* При повороте ключа они **раздвигаются**, образуя **вертикальный канал**
* Через этот канал мёд **стекает вниз**
* После этого ты возвращаешь ключ обратно → ячейки снова целые

📌 Это похоже на **водный шлюз**: ты открываешь дверцу → мёд вытекает → закрываешь → всё снова в порядке.

***

### 🗝️ 3. Работа ключа: что происходит при повороте?

#### Этапы:

1. Ты вставляешь ключ в боковую часть рамки
2. Поворачиваешь его на **90°**
3. Ключ активирует **центральный вал**, связанный с **тонкими лопастями**
4. Лопасти двигают половинками ячеек
5. Ячейки «раскрываются», образуют щели
6. Мёд начинает **медленно стекать вниз**
7. Он собирается в желобке и через трубку выходит в банку
8. После сбора ты возвращаешь ключ обратно → соты снова готовы к работе

📌 Это как **тайник, который открывается и закрывается**, чтобы мёд мог вытечь, а пчёлы не заметили изменений.

***

### 🍯 4. Как мёд стекает через Flow Frame?

#### Процесс:

1. 🐝 Пчёлы заполняют ячейки мёдом и **закрывают восковыми крышечками**
2. 🧠 Ты проверяешь уровень мёда через **прозрачное окно**
3. 🥄 Поворачиваешь ключ → ячейки раскрываются
4. 🍯 Мёд стекает по вертикальным каналам
5. 🕊️ Он собирается в нижнем желобке
6. 🧾 Через трубочку он вытекает в банку
7. 🧬 После этого ты возвращаешь ключ в исходное положение
8. 🥄 Пчёлы продолжают работать, как ни в чём не бывало

📌 Это гравитационная система, где **мёд течёт сам**, без насильственного отбора.

***

### 🧭 5. Почему пчёлы почти не реагируют?

| Фактор                          | Объяснение                               |
| ------------------------------- | ---------------------------------------- |
| 🐝 **Они живут внизу улья**     | Не видят, что сверху произошло изменение |
| 🧠 **Не теряют связи с маткой** | Матка в безопасности                     |
| 🕊️ **Не тревожатся от шума**   | Никто не открывает улей                  |
| 🧾 **Не теряют воск**           | Соты остаются целыми                     |
| 🧬 **Не меняют ориентацию**     | Запах улья не нарушается                 |
| 🥄 **Не останавливаются**       | Развитие семьи не прерывается            |
| 🌿 **Не покидают улей**         | Пчёлы остаются дома                      |

📌 Это улей, где **ничто не разрушается**, и **всё работает как часы**.

### ⚠️ 6. Возможные проблемы с Flow Frame

| Проблема                              | Причины                          | Как исправить                              |
| ------------------------------------- | -------------------------------- | ------------------------------------------ |
| 🧱 **Пластиковые соты деформируются** | От влаги или перегрева           | Используй деревянную основу                |
| 🥄 **Засор трубки слива**             | Воск, пыль, мусор                | Почисти после каждого сбора                |
| 🧾 **Мёд начал бродить**              | Если долго не собирать           | Собирай своевременно                       |
| 🧬 **Пчёлы игнорируют Flow Frame**    | Иногда предпочитают другие рамки | Убери лишние, оставь только Flow Frame     |
| 🧠 **Заедание механизма**             | Из-за влаги или загрязнения      | Смажь натуральным маслом                   |
| 🧾 **Нет сигнала, когда мёд готов**   | Требуется наблюдение             | Добавь цветовое окно или звуковой сигнал   |
| 🧬 **Не все породы принимают её**     | Нужно время адаптации            | Дай им пару недель на принятие             |
| 🥄 **Можно собрать жидкий мёд**       | Пчёлы ещё не запечатали ячейки   | Подожди, пока ячейки будут закрыты         |
| 🌿 **Может быть конденсат зимой**     | Если оставить рамку на холоде    | Убирай Flow Frame перед зимовкой           |
| 🧪 **Разные реакции от пчёл**         | Зависит от породы и климата      | Сделай несколько версий под разные условия |

📌 Эти слабые места помогут тебе понять, **что можно улучшить** и **как создать лучшую конструкцию**.

### 🧪 8. Как проверить Flow Frame?

| Шаг                                       | Что делать                                |
| ----------------------------------------- | ----------------------------------------- |
| 🛠️ **Проверь ключ**                      | Поверни — должен двигаться легко          |
| 🧽 **Протестируй трубку**                 | Пролей воду — должна стекать свободно     |
| 🪟 **Проверь прозрачное окно**            | Видно ли, как заполняется мёд             |
| 🧾 **Посмотри на пчёл**                   | Они должны подходить к рамке              |
| 🧬 **Проверь, нет ли воска внутри**       | Иногда пчёлы начинают строить соты внутри |
| 🥄 **Проверь защиту от грабителей**       | Леток должен быть закрыт                  |
| 🌳 **Проверь температурный режим**        | Пчёлы не должны перегреваться             |
| 🧠 **Проверь, нет ли брожения**           | Запах и цвет скажут об этом               |
| 🧾 **Проверь, нет ли клещей**             | Особенно весной и осенью                  |
| 🧬 **Проверь, нет ли паутины или мусора** | Очисти внутренние каналы                  |

📌 Главное правило:

> _«Не открывай улей часто. Только тогда, когда нужно»_

```
[Пчёлы достраивают соты]  
        ▼  
[Ячейки заполнены мёдом и запечатаны]  
        ▼  
[Ты поворачиваешь ключ]  
        ▼  
[Половинки ячеек раздвигаются]

        ▼

        ▼  
[Пчёлы снова наполняют соты]
```

📌 Это **не просто механизм. Это поток**, который **можно открывать и закрывать**, не нарушая жизни внутри.
